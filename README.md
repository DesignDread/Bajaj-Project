# IEEE CIET Website

Welcome to the official repository of the **IEEE CIET Website**, a dynamic platform developed to serve IEEE members and prospective members at **Chitkara Institute of Engineering and Technology (CIET)**. This website streamlines event management, communication, and membership engagement while showcasing IEEE activities and providing valuable learning resources.

## Introduction
The **IEEE CIET Website** is designed to enhance user experience for IEEE members and administrators. The platform supports automated event management, blog submissions, learning resources, and a chatbot to handle user queries. It also serves as a knowledge hub for prospective members, simplifying the process of joining IEEE through an interactive "Apply Now" feature.

## Features
Key features of the website include:

- **Home Page**:
  - Hero section with a "Join Now" button.
  - Display of upcoming events, sponsors, and the benefits of joining IEEE.

- **About Page**: Overview of the IEEE CIET chapter and its mission.

- **Events Page**: Lists past and upcoming events with event registration functionality.

- **Alumni Page**: Profiles and success stories of IEEE alumni members.

- **Contact Page**: Contact form and admin details.

- **Admin Dashboard**: Panel to manage events, blog posts, and other website content.

- **Blog Submission**: Users can submit blogs for approval by admins.

- **Event Notifications**: Automated emails and push notifications for event updates.

- **Learning Resources**: Access to IEEE technical materials and resources.

- **Chatbot**: AI chatbot for instant event-related queries and FAQs.

- **Photo/Video Gallery**: Visual highlights of IEEE events.

## Project Goals
- **Improve Communication**: Provide a centralized platform to share news, upcoming events, and key information with members.
- **Automate Event Management**: Streamline the event registration process with automated confirmation emails and reminders.
- **Increase Member Engagement**: Encourage interaction through blogs, FAQs, and a chatbot for instant queries.
- **Enhance Information Sharing**: Offer learning resources and promote the benefits of IEEE membership.
- **Promote Membership Growth**: Provide an easy "Apply Now" process to attract new members.

## Tech Stack
The project uses the following technologies:

- **Frontend**:
  - Next.js
  - React.js
  - TailwindCSS for styling
  - Three.js & GSAP for animations and visual effects

- **Backend**:
  - [IEEE Backend](https://github.com/Soham2395/IEEE-Backend)

- **Additional Services**:
  - Nodemailer or SendGrid for sending email notifications
  - Firebase Cloud Messaging (FCM) for push notifications
  - Vercel for hosting the frontend

## Installation
- To run this project locally, follow these steps:

### Prerequisites
- Ensure you have **Node.js** (v14 or above) installed.

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Soham2395/IEEE-Website.git
   ```

2. Navigate into the project directory and install dependencies:
   ```bash
   cd ieee-website
   npm install
   ```

3. Set up environment variables:
   ```bash
   NEXT_PUBLIC_API_URL= # Backend API URL
   ```

4. Run the frontend server:
   ```bash
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000`.

## Usage
Once installed, the following features will be accessible:

- **Home Page**: Displays upcoming events, IEEE benefits, and a "Join Now" section.
- **Admin Dashboard**: Admins can upload and manage content.
- **Event Registration**: Users can register for events and receive confirmation emails.
- **Blog Submission**: Users can submit blogs for approval.
- **Learning Resources**: Access to IEEE technical resources.
